Yuicompressor Online?

js,css 압축할때 매번 환경구성하기가 번거로움. 간단하게 api형태로 js파일을 던지면 압축된 결과를 리턴!!

사용방법

curl -F upfile=@test.js  http://{host}/compress

POST 파라미터

이름 설명 기본값 필수
upfile 업로드 파일
type Specifies the type of the input file (js|css) 파일확장자
charset Read the input file using charset utf-8
linebreak Insert a line break after the specified column number 1000
